Campus Event Management Hub

Overview

The Campus Event Management Hub is a comprehensive platform designed to simplify the organization and participation in university events. It offers secure login, personalized dashboards, event creation, registration facilities, and profile management for students and club administrators. The system fosters seamless communication and engagement within the campus community.

Technology Stack

  • Frontend: HTML, CSS, Bootstrap, JavaScript
  • Backend: Django (Python)
  • Database: SQLite3

Core Features

  • Role-Based Dashboards:Custom interfaces tailored for students and admin users.
  • Streamlined Event Enrollment:Easy browsing and registration for students.
  • Event Creation and Management:Admins can set up detailed events with various attributes such as title, description, date, imagery, and capacity.
  • Capacity & Status Monitoring:Event registration automatically closes when maximum capacity is reached, with real-time indicators showing whether events are "Open" or "Closed."
  • Event Organization:Events are categorized into upcoming and past events for easy navigation.
  • Additional Functionalities:
    • Registration confirmation via email
    • Countdown timers for upcoming events
    • QR code-based digital ticketing
    • Administrative approval workflow before publishing events
    • User-initiated event withdrawal options
    • Search and filter options for event discovery
  • Responsive Design:Fully adaptable layout optimized for devices of all sizes.

The system aims to facilitate efficient management and active participation in campus events, promoting community engagement and streamlined coordination among students and club organizations.
